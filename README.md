# Minimal Python AI Agent from Scratch

A lightweight, framework-free AI agent built using Python and OpenRouter. It implements the core **Thought → Action → Observation** loop, demonstrating manual tool calling, prompt engineering, and function execution without heavy dependencies like LangChain or LangGraph.

---

## How To Run

### 1. Set Up Your Environment Variable
Get an API key from [OpenRouter](https://openrouter.ai/). Create a `.env` file in the root directory of your project and add your key:

OPENROUTER_API_KEY=your_api_key_here
